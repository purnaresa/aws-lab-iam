# AWS Lab - IAM

This repository contains various use-cases and examples related to AWS Identity and Access Management (IAM) services.

## Use-Cases

- [IAM Access Analyzer Findings Exporter](access-analyzer/)

### [IAM Access Analyzer Findings Exporter](access-analyzer/)

This use-case demonstrates how to retrieve findings from an AWS IAM Access Analyzer analyzer and export them to a CSV file using Python and the AWS SDK for Python (Boto3).

The `access-analyzer` folder contains the Python script and related files for this use-case.
